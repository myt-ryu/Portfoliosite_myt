# ページ設計案

## 全体構成

シングルページアプリケーション（SPA）として構成し、スムーズなスクロールで各セクションへ遷移する。

---

## セクション構成

### 1. ヘッダー / ナビゲーション
- **固定ヘッダー**：スクロール時も上部に固定表示
- **ロゴ / 名前**：左側に配置
- **ナビゲーションメニュー**：右側に配置
  - About
  - Projects
  - Contact
- **モバイル対応**：ハンバーガーメニューに切り替え

---

### 2. ヒーローセクション
- **ファーストビュー**：画面全体を使用（100vh）
- **メインビジュアル**：
  - シンプルな背景（白 or 薄いグレー）
  - ミニマルなタイポグラフィで名前と肩書きを表示
- **キャッチコピー**：1〜2行の簡潔な自己紹介文
- **スクロールインジケーター**：下矢印アニメーション

---

### 3. About（自己紹介）セクション
- **レイアウト**：2カラム（デスクトップ）/ 1カラム（モバイル）
  - 左：プロフィール写真（円形 or 正方形）
  - 右：自己紹介テキスト
- **内容**：
  - 名前
  - 職種 / 専門分野
  - 経歴の概要（3〜5行程度）
  - スキルセット（アイコン + テキスト）
  - **SNSリンク**：アイコン形式で横並び表示
    - GitHub
    - LinkedIn
    - X（Twitter）
    - その他（Dribbble, Behance など任意）
- **デザイン**：
  - 余白を十分に取る
  - フォントは読みやすいサンセリフ系

---

### 4. Projects（プロジェクト一覧）セクション

#### 4.1 ピックアップ表示
- **カテゴリ別に代表作を1つずつ表示**
- **カテゴリ**（8種類）：
  - イベント/ワークショップのディレクション
  - イベント/ワークショップのファシリテーション
  - 執筆
  - プロジェクト企画・プロデュース
  - 講演会登壇
  - 研究
  - 広報・PR（プレスリリース、SNS運用）
  - その他
- **レイアウト**：2行 × 4列のグリッド表示（デスクトップ）
  - タブレット：2行 × 2列
  - モバイル：1列で縦に並べる
- **カードデザイン**：
  - サムネイル画像（アスペクト比 16:9 or 4:3）
  - プロジェクト名
  - 簡単な説明（1〜2行）
  - 使用技術タグ
- **ホバーエフェクト**：
  - 画像の拡大
  - オーバーレイで「View Project」ボタン表示

#### 4.2 More Projects ボタン
- **配置**：ピックアップカードの下、中央揃え
- **デザイン**：
  - ミニマルなボタンデザイン
  - 「View All Projects」または「More」
- **動作**：
  - クリックで全プロジェクト一覧ページへ遷移
  - または、アコーディオン/モーダルで追加表示

#### 4.3 プロジェクト詳細ページ（別ページ）
- **ヘッダー画像**：プロジェクトのメインビジュアル
- **コンテンツ**：
  - プロジェクト概要
  - 課題と解決策
  - 使用技術・ツール
  - プロセス（任意）
  - 成果・結果
  - リンク（GitHub / デモサイト等）
- **ナビゲーション**：前後のプロジェクトへのリンク

---

### 5. Contact（問い合わせ）セクション
- **シンプルな構成**：
  - セクションタイトル：「Contact」
  - 簡単なメッセージ：「お仕事のご依頼・ご相談はこちら」等
  - メールアドレス：クリックでメーラー起動（mailto:）
- **デザイン**：
  - 中央揃え
  - 十分な余白
  - メールアドレスは目立つように

---

### 6. フッター
- **内容**：
  - コピーライト表記
  - SNSリンク（GitHub, LinkedIn, X など）
- **デザイン**：
  - 最小限の要素
  - 背景色でセクション区切りを明示（薄いグレー等）

---

## デザインガイドライン

### カラーパレット
| 用途 | 色 |
|------|-----|
| 背景（メイン） | #FFFFFF（白） |
| 背景（アクセント） | #F5F5F5（薄いグレー） |
| テキスト（メイン） | #1A1A1A（ほぼ黒） |
| テキスト（サブ） | #666666（グレー） |
| アクセント | #000000（黒）または選定したブランドカラー |

### タイポグラフィ
- **見出し**：サンセリフ系（例：Inter, Helvetica Neue）
  - H1: 48px / 36px（モバイル）
  - H2: 32px / 24px（モバイル）
  - H3: 24px / 20px（モバイル）
- **本文**：16px / 1.6〜1.8 行間
- **フォントウェイト**：Regular (400), Medium (500), Bold (700)

### 余白・グリッド
- **コンテナ幅**：最大1200px
- **セクション間余白**：120px / 80px（モバイル）
- **グリッド**：12カラムグリッドシステム

### アニメーション
- **トランジション**：0.3s ease
- **スクロールアニメーション**：フェードイン、スライドアップ（控えめに）
- **ホバーエフェクト**：微細な拡大、色変化

---

## レスポンシブブレークポイント
| デバイス | 幅 |
|----------|-----|
| デスクトップ | 1024px 以上 |
| タブレット | 768px 〜 1023px |
| モバイル | 767px 以下 |

---

## 技術スタック
- **フレームワーク**：React + Vite
- **スタイリング**：CSS Modules または Tailwind CSS
- **アニメーション**：Framer Motion（任意）
- **ホスティング**：GitHub Pages
